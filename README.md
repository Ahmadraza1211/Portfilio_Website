# My Portfolio

Welcome to my personal portfolio website! This repository contains the source code for my portfolio, showcasing my skills, projects, and professional experience.

## Features

- **Modern UI**: Clean and aesthetically pleasing design.
- **Responsive**: Fully responsive and optimized for mobile, tablet, and desktop views.
- **Dynamic Content**: Highlights my latest projects, skills, and contact information.

## Technologies Used

- **React**: For building the user interface.
- **Vite**: For fast build times and local development.
- **CSS**: For styling and layout.

## Running Locally

To run this project locally on your machine, follow these steps:

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   ```

2. Navigate into the project directory:
   ```bash
   cd Portfilio_Website-main
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173`.
